## 步骤

```bash
# 1.安装node14.x（一定要控制在14.x，版本太高会出一系列幺蛾子），确保在 cmd 下 node -v 能出现版本号
# 2.克隆并进入项目目录
# 3.安装依赖
npm install --registry=https://registry.npm.taobao.org

# 4.启动服务
npm run dev

# 5.打包部署到生产
npm run build
```

浏览器访问 [http://localhost:8023](http://localhost:8023), 账号admin, 密码123456
